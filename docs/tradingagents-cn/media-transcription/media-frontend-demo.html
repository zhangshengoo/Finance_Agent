本版要点
  • 排版修复：浏览器主区改 flex + min-width:0，卡片内部纯块状纵向——不再出现标题竖排 / 大片空白。
  • 节点新格式：类型标签 + 精炼短语（粗体）+ 一两行专业细节，单条 ≤100 字。
  • 类型化：📌事实 · 🔗逻辑 · 💡观点 · 📈趋势 · ⚡操作 · ⚠风险（可扩展）；🔗 渲染因果链 A→B→C。
  • 立场萃取：📈趋势 + ⚡操作 节点 → 立场徽章 / 演化图，统一走 ②③，不依赖 wiki。

💰 黄金 · 白银（操作复盘）
📈 趋势黄金中长期看 5000–5200看多
破 4600 为关键确认位；无论盘中踩破 4600、还是回踩 4400–4000 区间，中长期看涨逻辑均不改变，只是节奏不同。
⚡ 操作黄金破 4600 → 加仓目标 5000-5200止损 4600
盘中冲破 4600 即在该位加仓，作为本轮加仓触发点；策略为分批越跌越买，回踩不破则继续持有。
⚠ 风险跌破 4600 且无法收回则破位
若回踩跌破 4600 且无法快速收回，中期看涨结构被破坏，应主动降杠杆规避，不再越跌越买。
⚡ 操作白银未破 72 → 持有止损 72
白银与黄金同向，止损严格设在 72；未跌破前维持现有杠杆持仓，一旦跌破即离场不恋战。
🏛 美债 · 全球化
🔗 逻辑全球化瓦解→财政赤字货币化→长端利率上行
去全球化削弱海外对美债的外部需求，财政赤字只能靠货币化承接，从而系统性推升长端利率，是本轮债市主线。
📌 事实30Y 美债收益率达 5%
30 年期美债收益率升至 5%，被视为本轮缩表空间见顶、长端定价压力外显的直接信号。
💡 观点美联储缩表概率极低美债看空
AI 资本开支正与国债争夺金融机构资金，流动性偏紧，联储难以主动缩表，长端利率易上难下。
🤖 AI 板块轮动
🔗 逻辑GPU→CPU/HBM→光模块→边缘
算力需求沿产业链向外溢出，资金从 GPU 依次轮动至 CPU/HBM、光模块，最终扩散到边缘侧。
💡 观点美股靠"应用层涨价"自证 AI
硬件叙事已充分定价，后续须靠应用层涨价兑现真实盈利，才能支撑 AI 故事的可持续性与估值。

02对着 Demo 逐块解释

内容来源： ①采集raw②预览 ③聚合timeline④wiki(降级·可选)⑤前端live
1

创作者头卡

头像/名称/UID + 统计（条目·视频·动态·活跃标的）。"活跃标的"= 有立场记录的标的数。

来源①名称/UID ＋ ③计数 ＋ ⑤去重
2

时间线（默认折叠·按行排版）

折叠态一行：时间 + 类型 + 标题 + 立场色点。展开后：元信息行 / 标题行 / 总结行 / 知识树 各自整行。

来源①时间/标题 ＋ ②一句话总结 ＋ ③立场色点
3

类型化知识点树（核心）

主题下挂带类型节点，每条 = 类型标签 + 精炼短语 + 一两行细节（≤100字）：📌事实 / 🔗逻辑(因果链) / 💡观点 / 📈趋势 / ⚡操作 / ⚠风险。视频内容是什么，就用对应类型装，类型集可扩展。

来源②raw-preview 产 typed nodes → ③结构化进 timeline → ⑤差异化渲染
4

立场徽章（自节点萃取）

把树里 📈趋势 + ⚡操作 节点（带方向/价位）聚合成立场。立场是知识树特定类型节点的投影，一份数据两处用，统一走 ②③。

来源②趋势/操作节点 → ③timeline.stances[] → ⑤渲染（④不参与）
5

观点演化图（按领域折叠）

领域 chips 折叠选择（默认黄金）。仓位/止损轴为默认，可切多空泳道。数据按标的聚合各条目立场成时间序列。

来源⑤stanceIndex 按标的聚合 ← ③各条目 stances[]（自趋势/操作节点）

03对应的 preview 模版（typed nodes）

每节点带 type + head(精炼短语) + detail(一两行)；趋势/操作节点带方向价位 → 自动萃取立场。逻辑节点用 chain 画因果链。值取自真实正文，缺失留空。

topics:
  - t: 黄金·白银（操作复盘）   icon: 💰
    nodes:
      - {type:trend,  target:黄金, dir:long, head:中长期看5000-5200, detail:破4600为关键确认位}
      - {type:action, target:黄金, action:加仓, target_price:'5000-5200', stop_loss:'4600', head:破4600加仓, detail:分批越跌越买}
      - {type:risk,   target:黄金, head:跌破4600且收不回则破位}
      - {type:action, target:白银, action:持有, stop_loss:'72', head:未破72持有}
  - t: 美债·全球化
    nodes:
      - {type:logic, chain:[全球化瓦解, 财政赤字货币化, 长端利率上行]}
      - {type:fact,  head:30Y美债收益率达5%}
      - {type:view,  target:美债, dir:short, head:缩表概率极低, detail:AI资本开支与国债争资金}
  - t: AI板块轮动
    nodes:
      - {type:logic, chain:[GPU, CPU/HBM, 光模块, 边缘]}
      - {type:view,  head:美股靠应用层涨价自证AI, detail:需应用层兑现盈利支撑估值}
# 立场 = 解析时从 type∈{trend,action} 且带 dir 的节点自动萃取，不重复手写

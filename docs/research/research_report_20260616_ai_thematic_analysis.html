Executive Summary

在AI驱动的投资研究领域，工具生态已分化为三条明确路径：开源多智能体框架（以 TradingAgents、FinRobot 为代表）提供可定制研究管道，最适合有 Python 基础的独立研究者；MCP 工具服务器（Alpha Vantage、EODHD、LSEG 等）将专业金融数据直接接入 Claude 等大模型，门槛低、整合快；国内商业投研平台（同花顺问财 2.0、东方财富妙想、雪球思考）面向 A 股投资者，提供从选股到产业链图谱的一站式 AI 服务。

对于"产业链分析 + 主题投资 + 个股挖掘"的具体需求，最高性价比的组合是：TradingAgents-CN + EODHD MCP + 东方财富妙想，配合知乎投资研究社区和硅谷101播客持续跟踪。

重点推荐：TradingAgents-CN A股适配

TradingAgents-CN 是目前对国内投资者最友好的开源多智能体框架。相比原版 TradingAgents，它完成了关键的本地化改造：接入 AKShare 和 Tushare 提供 A 股实时数据，支持 DeepSeek、通义千问、智谱 GLM 等国内 LLM，输出中文分析报告，并提供 Docker 容器化部署方案。最新版架构已升级为 FastAPI + Vue 3 + MongoDB/Redis，性能提升约10倍。[3]

框架的内置新闻质量过滤机制有效减少噪音干扰，对于主题投资中的信息甄别尤为重要。建议配合 EODHD MCP 数据层以弥补实时行情覆盖的不足。

四、评测基准与竞赛社区
FinBen — 最全面的 LLM 金融评测基准 [19]

FinBen 收录 42 个数据集、24 个金融任务，覆盖信息提取、文本分析、问答、文本生成、风险管理、预测、决策和双语测试。NeurIPS 2024 发布的评测结果显示：GPT-4 在信息提取和股票交易上领先；Gemini 在文本生成和预测上较强；所有模型在高级任务（复杂推理、投资决策）上仍有显著局限。选择框架底层 LLM 引擎前应参考此基准。

Open FinLLM Leaderboard [20]

持续更新的金融 LLM 能力排行榜，追踪各大模型在金融任务上的实时表现，是选择基础模型的快速参考。

FinAI Contest 2025（IEEE CSCloud 2025）[21]

每年举办，4个赛道：股票交易智能体（FinRL-DeepSeek）、加密货币交易、金融分析（FinGPT实战）、DeFi流动性提供。竞赛 GitHub 仓库中的获奖方案是产业链分析方法论的重要学习资源，汇聚学术界与工业界金融AI研究者。

关键洞察
  • 国内/国际分轨: 国内平台（妙想、问财）在 A 股数据时效性上无可替代；开源框架（TradingAgents-CN）在定制化深度上占优。两者结合是最佳实践，不必二选一。
  • MCP 层是关键粘合剂: MCP 服务器让任何 AI 助手（Claude、GPT）都能实时获取专业金融数据，是连接智能体推理能力和数据源的最优路径。无论使用哪个框架，优先配置合适的 MCP 数据层。
  • 主题分析难点在"传导链": 现有工具在"识别主题受益股"上能力参差不齐。东方财富妙想的事件传导图和天风证券研报是目前最可靠的中文产业链传导分析来源，可作为 AI 分析结果的人工验证基准。
  • 个股挖掘需多信号交叉验证: 基本面 + 情绪 + 技术三类信号都需交叉验证。TradingAgents 的多智能体架构天然支持这种协作分析范式，单一信号的误判率显著高于三信号交叉。

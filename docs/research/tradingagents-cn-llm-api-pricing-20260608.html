0 · 结论速览

最省钱组合：智谱 GLM-4-Flash（免费）→ DeepSeek deepseek-chat（¥1/¥2）→ 通义 qwen-turbo（¥0.3/¥0.6）。国产模型性价比远超海外。
国内首选渠道：要 Claude/GPT 又只有人民币 → AIHubMix（项目官方推荐/赞助，支付宝充值、OpenAI 兼容）；只用国产 → 硅基流动或百度千帆。

6 · 给 TradingAgents-CN 用户的配置建议

  • 零成本起步：GLM-4-Flash(免费)或硅基流动免费模型先跑通多智能体流水线。
  • 低成本主力(国内直连)：DeepSeek deepseek-v4-flash(别再写 deepseek-chat，7-24后失效) ¥1/¥2，或 qwen-plus；深度推理用 deepseek-v4-pro/qwen3-max。
  • 要用 Claude/GPT：走 AIHubMix，.env 填 AIHUBMIX_API_KEY + AIHUBMIX_BASE_URL=https://aihubmix.com/v1，支付宝充值免海外卡。
  • 成本控制：开缓存(命中省90%+)、大批量用 Batch 5折；订阅只对重度高频的智谱 Coding Plan 才划算。
  • 别买消费级订阅当 API：ChatGPT Plus / Claude Pro 不能驱动本项目。

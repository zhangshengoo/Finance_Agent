## 二、各数据源详细说明

### 1. BaoStock（证券宝）

**定位：** 纯免费、无需注册的 A股历史行情库，适合量化入门和学术研究。

**覆盖维度：**
- 日/周/月 K 线（1990-12-19 至今）
- 分钟 K 线（5/15/30/60 分钟，1999 年至今）
- 季频财务数据（资产负债、利润表、现金流、杜邦指标，2007 年至今）
- 沪深 300、上证 50、中证 500 等指数成分股

**不覆盖：** 资金流向、北向资金、龙虎榜、融资融券、情绪/舆论、新闻公告、研报。

**Python SDK：** 官方提供 `baostock` 包，`pip install baostock`，返回 pandas DataFrame。

**适合人群：** 个人量化入门、学术回测。需要纯净历史行情但不需要另类数据的场景。

**局限性：** 数据停留在收盘后日频，无实时行情，功能单一，长期维护活跃度一般。

### 2. AKShare

**定位：** 最全面的免费 A股开源数据库，本质是将东方财富、同花顺、新浪财经等网站的公开接口封装成 Python 函数。

**覆盖维度（特别详细）：**

**资金流向：**
- `stock_fund_flow_em()` — 东财个股资金流（主力/超大单/大单/中单/小单 净流入）
- `stock_sector_fund_flow_hist()` — 行业历史资金流
- `stock_concept_fund_flow_hist()` — 概念板块历史资金流
- 同花顺大单追踪接口

**北向资金/陆股通：**
- `stock_hsgt_fund_em()` — 沪深港通资金实时流向
- 陆股通个股持股明细、机构排行、分时数据

**龙虎榜：**
- `stock_lhb_detail_em()` — 龙虎榜每日明细（东财数据源）
- 个股上榜统计、机构买卖统计、机构席位追踪、营业部排行
- 营业部实力排名、每日汇总

**融资融券：**
- `stock_margin_detail_szse()` / `stock_margin_detail_sse()` — 深沪交所融资融券明细
- 融资融券汇总、标的证券名单及保证金比例

**情绪/舆论/热榜：**
- `stock_hot_rank_em()` — 东财人气榜、飙升榜（A股、港股）
- 雪球关注/讨论/交易排行榜
- 互动易问答数据、上证 e 互动
- 个股人气实时排名、热门关键词、热搜股票

**新闻公告：**
- `stock_news_em()` — 个股新闻（东财）
- 沪深京 A股公告、巨潮资讯公告

**研报/评级：**
- 个股券商研究报告摘要
- 机构推荐评级记录
- 盈利预测（东财、同花顺）

**定价：** 完全免费，MIT 开源协议。

**Python SDK：** `pip install akshare`，函数风格简洁，返回 DataFrame。

**数据质量/实时性：** 基于网页爬虫，稳定性依赖源网站（东财/同花顺）。行情数据一般为 T+1，部分接口支持日内延迟行情（非 Level-2）。爬虫有被限速/封禁风险，高频调用需节制。

**适合人群：** 个人投资者、量化研究入门、构建简单选股系统。不适合需要毫秒级实时数据或高并发生产环境的场景。

**已知问题（截至 2026 年）：** 部分接口因源站反爬升级偶有失效；建议在关键业务中加熔断逻辑。

### 3. pysnowball（雪球非官方 Python API）

**定位：** 第三方开源库，逆向封装雪球 APP 的 HTTP 接口，主要获取雪球平台的社交/情绪数据。

**覆盖维度：**
- 个股实时报价（来自雪球展示层）
- 投资者持仓及组合调仓记录
- 热帖/讨论热度（关注数、讨论数、交易数）
- 现金流量、财务指标（F10 数据）
- 机构持仓（部分）

**不覆盖：** 资金流向（主力/大单）、北向资金、龙虎榜、融资融券、研报评级。

**定价：** 完全免费，但需要自行从浏览器获取 `token`（cookie 中提取）。

**Python SDK：** `pip install pysnowball`，调用前 `ball.set_token('你的token')`。

**数据质量/实时性：** 依赖雪球 API 稳定性，雪球无官方开放协议，接口随时可能被修改或限速。仅适合低频个人使用。

**适合人群：** 需要社交情绪维度补充（雪球讨论热度、大 V 观点追踪）的个人投资者。

### 4. Mairui（麦蕊智数）

**定位：** 国内轻量级股票数据 SaaS，提供免费证书入门，覆盖行情和部分资金数据。

**覆盖维度：**
- A股实时行情（沪深）
- 技术指标（MACD、KDJ、均线等）
- 个股资金流向（主力净流入排名、大单排名）
- 涨跌停池、板块数据

**不覆盖：** 北向资金实时、龙虎榜详情、融资融券、情绪/舆论、新闻、研报。

**定价：** 免费证书 300 req/min，付费升级后频率更高，具体套餐见官网 `mairui.club`。

**Python SDK：** 纯 HTTP REST API，JSON 响应，无官方 Python 包装，需自行封装。

**数据质量/实时性：** 中等，日内延迟行情，非 Level-2。

**适合人群：** 轻量级个人项目、快速原型验证。

### 5. iTick

**定位：** 新兴 A股/港股/美股实时行情 SaaS，主打低延迟 Tick 数据。

**覆盖维度：**
- A股/港股/美股实时报价、五档盘口（Level-2 可选）
- 分钟 K 线、历史 K 线（15 年+）
- 龙虎榜数据
- 财务报表

**不覆盖（明确）：** 主力资金流向、北向资金、融资融券、舆论情绪、研报评级、新闻公告。

**定价：** 基础功能免费，高级接口（Tick 深度/Level-2）按调用次数付费，具体见 `docs.itick.org`。

**Python SDK：** REST + WebSocket 双协议，支持 Python/Java/Go，有官方文档（旧文档已停维护，参见 `docs.itick.org`）。

**数据质量/实时性：** 优秀，延迟 <100ms，通过全球节点加速。

**适合人群：** 需要低延迟实时行情的个人开发者或小团队，不需要复杂另类数据。

### 6. Tushare Pro

**定位：** 国内最主流的免费+积分制 A股数据平台，数据覆盖广、文档完善，是个人量化研究首选。

**积分制度（关键）：**

|---------|---------|---------|
| 0～119 | 受限 | 注册即得 |
| 120+ | 50次/分钟 | 基础数据开放 |
| 2,000+ | 提升 | 约 ¥200 捐助兑换 |
| 5,000+ | 更高 | 约 ¥500 捐助兑换 |
| 8,000+ | 最高 | 约 ¥800 捐助兑换 |

> 注：积分有效期 1 年，到期后重新购买。企业用户费用为个人 10 倍。

**覆盖维度：**

**资金流向（需 5,000 积分）：**
- `moneyflow_dc` — 东财个股资金流（日频，主力/大单/中单/小单 净流入，收盘后入库）

**北向资金：**
- 沪深股通资金流向、持股明细（部分接口需 2,000 积分）

**龙虎榜（需 2,000 积分）：**
- `top_list` — 龙虎榜每日明细（2005 年至今）
- `top_inst` — 龙虎榜机构交易明细

**融资融券（需 2,000 积分）：**
- 融资融券每日汇总、标的证券信息

**情绪/舆论（需高积分）：**
- `dc_hot` — 东财 APP 热榜（需 8,000 积分）
- `ths_hot` — 同花顺 APP 热榜（需 5,000+ 积分）
- `kpla` — 开盘啦涨停榜单

**新闻公告：**
- 上市公司公告（部分需积分）

**研报/评级：**
- 分析师评级、盈利预测（需 2,000+ 积分）

**Python SDK：** 官方 `tushare` 包，`import tushare as ts`，返回 DataFrame，接口文档完善。

**数据质量/实时性：** 良好，以日频为主，收盘后 T+0 入库（一般 15:00～16:00）。实时 Tick 接口（爬虫）0 积分可访问，但稳定性较差。

**适合人群：** 个人量化研究、策略回测、轻量数据管道，是免费+低成本的最优综合选择之一。

**定位：** 在线量化平台，提供完整的数据+回测+实盘环境，数据以平台内调用为主，也提供离线 SDK。

**覆盖维度：**
- 完整 A股行情（日/分钟）、指数、期货、基金
- 资金流向（主力净流入、大单）
- 北向资金
- 融资融券
- 分析师预测、评级
- 外部情绪数据（另购）
- 财务数据（2005 年至今）

**定价：**
- 在线平台：新用户 6 个月免费额度（有调用上限）
- 离线 JQData SDK：个人版约 ¥3,000～¥6,000/年（参考，官方未公开定价，需询价）

**Python SDK：** `jqdatasdk`，离线使用需授权，平台内使用不受限。

**数据质量/实时性：** 良好，日频+分钟频，数据从 2005 年起，经过清洗，质量较高。复权因子处理完善。

**适合人群：** 个人量化策略开发者（在线平台入门首选），需要一站式数据+回测环境的个人或小机构。

**局限：** 离线 SDK 与平台深度绑定，独立使用成本较高；情绪/舆论数据需额外购买。

**适合人群：** 个人进阶量化研究者、私募基金研究员，尤其适合需要另类数据（情绪、ESG）的多因子选股策略。

### 11. Wind（万得）

**定位：** 中国金融数据行业标杆，覆盖最全、质量最高，是绝大多数国内机构的第一数据源。

**覆盖维度（最全）：**
- 所有市场全品种完整行情（A股/港股/美股/期货/期权/债券/基金/汇率）
- **个股资金流向**（主力/大单，实时及历史）
- **北向资金**（实时、日内分时、持股明细、机构排行）
- **龙虎榜**（完整历史，2003 年至今）
- **融资融券**（完整历史，实时汇总）
- 财经新闻、实时公告
- **完整研报**（正文可查，非摘要）、**分析师评级**（覆盖全市场）
- 宏观经济、行业数据、另类数据（舆情指数等）
- ESG、法律诉讼、股权结构等深度数据

**定价：** 个人量化版免费（功能受限）；标准机构版 ¥19,800～¥39,800/年/账号（参考，2023 年报道，近两年涨幅明显），AMS 资产管理版 ¥58,000+/账号/年。

**Python SDK：** `windpy`（需安装 Wind 终端），函数前缀 `w.`，文档完整，也提供 HTTP API（WDS）。

**数据质量/实时性：** 行业顶级，实时性最佳，数据全面且经过严格质控，研报为正文级别。

**适合人群：** 公募基金、券商、大中型私募、研究机构。个人用户因价格门槛过高，通常不考虑。

## 五、注意事项

1. **价格均为参考值**，商业产品（Choice/iFinD/Wind）均有谈判空间，实际成交价因采购数量、合同期限差异较大，以官方报价为准。

2. **免费爬虫接口风险**：AKShare、pysnowball 等依赖逆向或网页爬虫，源站反爬升级可能导致接口随时失效。生产环境建议加熔断/回退逻辑，并关注 GitHub Issues。

3. **Tushare 积分有效期**：积分 1 年到期，高积分接口（5,000+）单次成本约 ¥500，是个人用户性价比最高的付费路线之一。

4. **雪球非官方 API 法律风险**：雪球无官方开放协议，高频访问可能触发封禁或法律风险，仅建议低频个人使用。

5. **实时性层次**：免费源（AKShare/BaoStock）通常为 T+1 或日内延迟；付费商业数据（Choice/iFinD/Wind）可达分钟级甚至秒级；Level-2 逐笔成交数据需向交易所授权，AKShare 等无法覆盖。

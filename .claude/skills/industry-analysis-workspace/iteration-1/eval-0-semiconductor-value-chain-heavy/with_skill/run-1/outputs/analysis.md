### 1.2 景气与政策叙事（定性）

- **国产替代是十年级主线**：受外部出口管制（先进制程设备、EDA、高端 GPU/HBM）持续压制，"自主可控"从产业话题升级为国家战略，国家集成电路产业投资基金（"大基金"）多期落地，资本开支向本土设备/材料倾斜。
- **AI 算力拉动结构性需求**：AI 训练/推理对先进逻辑、HBM、先进封装（CoWoS/2.5D）需求爆发，国产算力链（GPU/ASIC/先进封装）成为新增量叙事。

- **风险面**：制程/设备代差仍在、设计端高端算力受限、晶圆扩产带来阶段性供给过剩风险。

## 2. 产业链结构（Value Chain）

半导体是典型**垂直行业**，价值沿"设备材料 → 设计制造封测 → 终端应用"链条分布（选型依据：cn-playbook §4，垂直行业用价值链图）。`|bottleneck` 标"卡脖子/高壁垒/国产化率低"环节。

### 上游 — 设备与材料（卡脖子最密集）
- **光刻机（尤其 EUV）|bottleneck** —— 国产化率最低、被出口管制核心环节
- **刻蚀 / 薄膜沉积（CVD/PVD）/ 清洗设备** —— 国产替代推进最快的设备环节
- **光刻胶（高端 KrF/ArF）|bottleneck** —— 高端胶仍高度依赖进口
- **大硅片（12 寸）/ 电子特气 / CMP 抛光材料** —— 材料国产化爬坡中
- 上游对应 A 股龙头：北方华创、中微公司、拓荆科技（设备）；沪硅产业、华特气体（材料）。

### 中游 — 设计 / 制造 / 封测
- **芯片设计（Fabless / IDM）** —— 数字/模拟/功率/存储多赛道，国产设计公司数量庞大
- **晶圆制造（Foundry）|bottleneck** —— 先进制程受设备管制，是国产链最大瓶颈与最大看点
- **封装测试（含先进封装 CoWoS/2.5D）** —— 国产封测全球份额较高，先进封装是 AI 链新增量
- 中游对应龙头：中芯国际、华虹公司（制造）；韦尔股份、兆易创新、卓胜微（设计）；长电科技、通富微电（封测）。

### 下游 — 终端应用
- **AI 算力（GPU / ASIC / HBM）** —— 当前最强增量叙事
- **消费电子（手机/PC）** —— 传统基本盘，周期复苏弹性来源
- **汽车电子（功率/MCU/SiC）** —— 电动智能化驱动的长期增量
- **工业控制 / 物联网** —— 稳定基本盘

> 产业链信息核心：**瓶颈集中在上游高端设备/材料（EUV、高端光刻胶）与中游先进制程晶圆制造**。国产替代的"易→难"梯度大致是：封测 → 成熟制程设备/材料 → 设计 → 先进制程制造 → EUV 光刻。

## 5. 投资含义（Investment Implications）

> 先做护城河评估（cn-playbook §3），再导出多空。

### 5.1 核心龙头护城河评估

**中芯国际（制造端）**

| 护城河 | 评级 | 说明 |
|---|---|---|
| 网络效应 | 弱 | 制造非平台生意 |
| 切换成本 | 中 | 客户产线验证周期长，一旦导入有黏性 |
| 规模经济 | 强 | 晶圆制造重资产，最小有效规模极高，国内唯一够格者 |
| 无形资产 | 中 | 制程 know-how + 牌照/政策卡位；但先进制程受设备管制，上限被外部锁住 |

**北方华创（设备端）**

| 护城河 | 评级 | 说明 |
|---|---|---|
| 网络效应 | 弱 | 设备非平台 |
| 切换成本 | 强 | 设备进客户产线需长周期验证，替换成本极高 |
| 规模经济 | 中 | 平台化多品类摊薄研发，规模优势成型中 |
| 无形资产 | 强 | 多品类设备专利 + 国产替代政策红利 + 客户验证壁垒 |

**护城河综合三要素**：
1. **持久优势**：设备端（北方华创/中微）= 切换成本 + 无形资产（验证壁垒+政策）；制造端（中芯）= 规模经济（国内唯一够格的先进制程载体）。
2. **结构性弱点**：制造端先进制程上限被海外设备管制锁死（自身无法修复）；设计端格局分散，单赛道难形成寡头护城河。
3. **现状 vs 趋势**：设备/材料国产替代护城河**在变宽**（订单+政策双驱动）；先进制程制造护城河受外部约束**短期难拓宽**。

### 5.2 最佳风险回报机会

**上游设备 / 材料 > 中游制造 > 设计**：设备端国产替代确定性最高、毛利率最优、政策弹性最直接，且切换成本护城河最硬。制造端是"国运赛道"但受外部管制掣肘、估值已充分；设计端机会分散、需个股精选。

### 5.3 主题性投注

- **国产替代深化**：设备 → 材料 → EDA → 先进制程的"难度阶梯"逐级攻坚，每一格突破都是催化。
- **AI 算力国产化**：国产 GPU/ASIC + HBM + 先进封装（Chiplet/CoWoS），是估值最有想象空间、也最受情绪驱动的支线。

### 5.4 多空争论

**牛方（bull）**：
- 国产替代是十年级、政策托底的确定性主线，设备/材料端订单与渗透率持续上行。
- AI 算力爆发为先进封装/HBM/国产算力链打开第二增长曲线。
- 设备龙头具备"切换成本+验证壁垒+政策红利"的硬护城河，强者恒强。
- 周期底部（静态 PE>TTM PE），盈利修复弹性大。

**熊方（bear）**：
- 估值处全市场极端高位（PE 第 30、PB 第 31、股息第 31），安全边际极低，复苏证伪即 PE/PB 双杀。
- 先进制程受海外设备/EDA 管制，制造端护城河上限被外部锁死，非自身可修复。
- 设计端格局分散、价格战频发，多数公司难有持久超额利润。
- 晶圆扩产潮存在阶段性供给过剩、周期下行风险。

### 5.5 催化剂（未来 3–6 月）

- 大基金/地方基金新一轮投向、国产设备/材料中标与验证通过公告。
- 国产算力链（GPU/HBM/先进封装）量产/客户导入进展。
- 半导体季度库存去化 + 产能利用率回升数据（具体值 `[UNSOURCED]`）。
- 外部出口管制升级 / 缓和 —— 双向催化，是本行业最大的外生变量。

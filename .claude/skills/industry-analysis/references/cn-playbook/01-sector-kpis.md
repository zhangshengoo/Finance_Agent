# §1 行业关键指标选取（申万行业 → 行业特有 KPI）

## 为什么需要这张表

龙头横评（Step 3）如果只摆「市值 / PE / 涨跌」，每个行业长得一样，看不出谁强谁弱。每个行业真正被投资者和经营者拿来衡量的，是**行业特有指标**——半导体看产能与国产化率，银行看息差与不良，白酒看吨价与预收款。先选对 3–5 个 KPI，再去填龙头表，分析才有信息量。

**用法**：在 Step 3 渲染 `### 3.1 龙头公司画像` 表之前，先按本行业从下表取 3–5 个 KPI，作为「差异化定位」列的展开依据（或在正文 3.2 加一段 KPI 对比）。表里没有的行业，自己挑「投资者和经营者实际 benchmark 的指标」。

## 申万一级 → 关键指标对照

| 申万行业（示例） | 行业特有 KPI（3–5 个） | 数据可得性 |
|---|---|---|
| 半导体 / 电子 | 产能利用率、国产化率、晶圆/封测良率、设备国产替代率、研发费用率 | 多为定性/年报；财务字段 P2 |
| 银行 | 净息差(NIM)、不良率、拨备覆盖率、核心一级资本充足率、ROE | 可得（P2 财务） |
| 白酒 / 食品饮料 | 吨价、预收款(合同负债)、经销商数量、毛利率、动销/库存周转 | 预收款=合同负债，可得 |
| 医药生物 | 研发管线数、研发费用率、集采暴露占比、创新药收入占比、毛利率 | 管线定性；财务 P2 |
| 新能源（光伏/锂电） | 单位成本(元/W·元/Wh)、产能(GW/GWh)、产能利用率、技术路线占比、毛利率 | 产能可得；成本需研报 |
| 新能源车 | 销量、单车均价(ASP)、毛利率、电池自供率、订单/交付 | 销量可得 |
| 房地产 | 销售面积、去化率、净负债率、现金短债比、拿地强度 | 财务 P2 |
| 券商 / 非银 | 净资本、ROE、经纪/自营/投行收入结构、市占率 | 可得 |
| 计算机 / 软件 | 营收增速、毛利率、研发费用率、人均创收、订单/合同负债 | 可得 |
| 机械 / 工业 | 订单(在手订单)、产能利用率、毛利率、出口占比、客户集中度 | 订单定性 |
| 公用事业 / 电力 | 装机容量、利用小时数、上网电价、燃料成本、ROE | 可得 |

> 缺失的 KPI 一律标 `[UNSOURCED]` 或 `null`，**不许估算**（与 Guardrail #4 一致）。KPI 是定性叙事时，放进「差异化定位」一句话里即可，不强求进表。

## 落到输出

- KPI 选取结论写进 sector 分析 markdown 的 `### 3.2 竞争动态` 或龙头表的「差异化定位」列。
- 若某 KPI 是可枚举数值且前端要展示，按 [frontend-kb-binding](../../../../docs/frontend-kb-binding.md) §2.1 的字段约定上浮（目前前端 sector 模版只固定解析龙头表，额外 KPI 走正文）。
